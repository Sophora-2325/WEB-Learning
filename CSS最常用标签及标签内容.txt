一、独一档的地位：可以设定容器为弹性箱子-display:
容器内物品的展示方式，决定展示方式，独占一块一行，还是在别人内部，还是被弹性箱子控制

二、控制大局观：
1、flex-direction:
弹性箱子内弹出的方向，水平还是垂直
2、justify-content:
管理弹性箱子内部件在主轴上的对齐方式
3、align-items:
管理弹性箱子内部件在副轴（交叉轴）上的对齐方式
4、flex-wrap:
管理内部件的换行方式
5、align-content:
用来管理内部件的换行的对齐方式

三、微调小元素：
order:
管理弹性盒子的子元素排列顺序
align-self：
管理弹性元素自身在副轴上的对齐方式
flex：
管理弹性元素如何分配空间


text-align:
文本的对齐方式
border:
设定箱子边框的厚度，颜色
其所延申的边框属性属于美化内容

margin:
容器盒子的外间距
padding:
容器盒子的内间距


appearance:
可以消除元素的自带样式

outline：
消除元素的焦点外边框

关于表格的小知识：
表格之间可以用
1.colspan（跨列合并）或 
2.rowspan（跨行合并）

关于背景图的一些设定
  background-image: url('背景图.jpg');
  /* changed code */
  background-size: contain;        /* 完整显示，不裁剪；可能有空白 */
  background-position: center;     /* 居中显示 */
  background-repeat: no-repeat;    /* 不重复平铺 */


flex: 0 0 40% 是 flex 简写属性，分别表示三个值：

各部分含义：

属性	值	说明
flex-grow	0	不扩张 — 当容器有剩余空间时，该元素不占用额外空间
flex-shrink	0	不收缩 — 当容器空间不足时，该元素不被压缩
flex-basis	40%	基础宽度 — 元素的初始宽度为父容器的 40%
